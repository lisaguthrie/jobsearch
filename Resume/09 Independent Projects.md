## INDEPENDENT PROJECTS

**Independent Developer** -- Redmond, WA *(January 2026 – Present)*
Developing open-source tools at the intersection of AI and K-12 governance and accessibility, driven by direct experience as a school board director and parent of a disabled child. Code available at github.com/lisaguthrie.

- **Worksheet Accessibility App Builder:** Converts scanned worksheets into accessible single-page web apps, customized for a child's specific barriers and learning needs. Built with Python/Claude API using multi-modal inputs (PDF/image); architecture separates generalizable design patterns from child-specific profiles to enable reuse across students.
- **School Improvement Plan Dashboard:** Transforms hundreds of pages of unstructured district PDFs into structured, searchable data with AI-powered Q&A, enabling board directors to shift SIP review from compliance exercise to meaningful oversight. Built with Python/React/Claude API; implemented prompt caching for ~90% cost reduction on repeat runs.
- **WA State Education Data MCP Server:** Provides LLM-friendly access to Washington State standardized assessment data for multi-district analysis. Built with Python/MCP protocol; compatible with Claude Desktop and other MCP clients.
- **WA Legislature Bill Alerts:** Azure Function that monitors Washington State Legislature floor activity and sends SMS notifications when a configured bill is scheduled, created after personally watching hours of legislative proceedings waiting for a school district bill to be heard. Built with Python/Azure Functions/Twilio.
